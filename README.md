# TINYSN74
Library for Texas Instruments SN74HC595 Shift Registers (and clones),
for use with ATTINY chips.

Communication is currently bit-banging, with SPI in the works.  

## Details
- Current version: 0.0.1
- Supports:
..- ATTinyx5 (25/45/85)
